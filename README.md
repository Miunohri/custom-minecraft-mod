# Minecraft Mod

A Java-based Minecraft mod for forge that expands gameplay by adding custom items,
armor sets, crafting recipes, and new mechanics.

## Features

### Custom Items
- Added new materials and items used for crafting
- Registered custom items through the Minecraft modding API

### Custom Armor Sets
- Created custom armor sets with unique materials and attributes
- Implemented armor registration and equipment functionality

### Custom Crafting System
- Added recipes for newly introduced items and armor
- Integrated new materials into progression mechanics

### Mob Drops
- Modified mob loot behavior to introduce new item drops
- Created custom drop mechanics for gameplay progression

## Technologies
- Java
- Minecraft Forge
- Gradle
- Object-Oriented Programming
